## 项目简介
该项目**复现**了下面B站视频的代码。
这是一个用于在特殊场合，如情人节表白，通过多张合照生成一段浪漫文字的项目。通过该工具，你可以选择一系列的照片，然后生成一段表白文字，让你的表白更加特别和独特。
